Sera AI - Smart Retail Assistant

Sera AI is a smart retail assistant that helps businesses understand sales,
predict demand, recommend products, and detect risks using AI logic.

## Features
- Sales insights
- Demand prediction
- Product recommendations
- Risk detection
- Voice interaction
- Chat-based AI assistant

## Technology Used
- HTML
- CSS
- JavaScript
- AI logic (rule-based + simulation)

## How to Run
1. Download the project
2. Open index.html in browser
3. Start chatting with Sera AI

## Use Case
Retail shop owners can use Sera AI to make better business decisions.
